# Учебный проект. Vortex: освоение FPGA-стека

## Тестирование функции умножения матриц (sgemm)

При тестировании использовались:
- Vortex 2.2 
- Docker Engine 29.5.2

### Структура репозитория
    data/
        ├── result/             # Результаты тестов
        ├── input/              # Входные данные для тестов
        └── errors/             # Примеры ошибок
    docs/             # Документация
        ├── docker/             # Файлы для сборки Docker-образа
        ├── vortex_doc/         # Документация по тестированию и симуляции в Vortex
        └── analyze.md          # Анализ результатов тестирования
